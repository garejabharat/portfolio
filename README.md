# Bharat Gareja — React Developer Portfolio

Modern, responsive portfolio built with React, Framer Motion, and Tailwind-inspired CSS.

## 🚀 Tech Stack
- React 18
- Framer Motion (animations)
- React Icons
- React Scroll (smooth navigation)
- React Type Animation (hero typewriter)

## 📦 Setup

```bash
npm install
npm start
```

## 🌐 Deploy to Vercel

1. Push this repo to GitHub: `https://github.com/garejabharat/portfolio`
2. Go to https://vercel.com/bharatgarejas-projects
3. Click "New Project" → Import from GitHub
4. Select this repo → Click "Deploy"

Done! Your portfolio is live. ✅
